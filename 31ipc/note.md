# 使用信号量实现互斥
父进程：打印O -> 睡眠一段时间 -> 打印O
子进程：打印X -> 睡眠一段时间 -> 打印X

```
父进程            子进程
P    -<-          P    -<-
打印O  |          打印X  |
sleep  |          sleep  |
打印O  |          打印X  |
V    ->-          V    ->-
```
在终端中显示的结果，以OO和XX形式出现，不会出现单独的O或X

注意点：
- 每次使用printf打印字符的时候，需要立即刷新到stdout，fflush(stdout)，否则可能会出现单独的O或X
